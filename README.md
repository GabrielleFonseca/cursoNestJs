# Projeto do curso de NestJs

Curso: [NestJS do zero com TypeORM, Mongoose, Prisma e Swagger](https://udemy.com/course/nestjs-do-zero/).

## Versions

**Node**: 20.9.0

**PNPM**: 9.1.2

## Installation

```bash
$ nvm install 20.9.0
```

```bash
$ nvm use 20.9.0
```

```bash
$ npm install -g pnpm
```

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
